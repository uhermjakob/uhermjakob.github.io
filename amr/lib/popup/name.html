:name   (AMR annotation guidance)

The role :name is used for named entities:

Example:  "Steven Spielberg lives in Los Angeles ."
     (l / live-01
           :ARG0 (p / person :wiki "Steven_Spielberg"
                 :name (n / name :op1 "Steven" :op2 "Spielberg"))
           :location (c / city :wiki "Los_Angeles"
                 :name (n2 / name :op1 "Los" :op2 "Angeles")))

Example:  "The USS Iowa passed under the Golden Gate Bridge ."
     (p / pass-02
           :ARG0 (s / ship :wiki "USS_Iowa_(BB-61)"
                 :name (n / name :op1 "USS" :op2 "Iowa"))
           :ARG1 (u / under
                 :op1 (b / bridge :wiki "Golden_Gate_Bridge"
                       :name (n2 / name :op1 "Golden" :op2 "Gate" :op3 "Bridge"))))
