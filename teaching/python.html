  Regular Expressions

    Basics
    • \d   Matches any digit, e.g. 0-9, but also digits in foreign scripts     Example

              
      import re
s = "I bought 10 pounds of potatoes for $5.27."
re.findall(r'\d', s)
      ['1', '0', '5', '2', '7']
    • \d+   Matches one or more digits     Example

              
      import re
s = "I bought 10 pounds of potatoes for $5.27."
re.findall(r'\d+', s)     # + indicates "one or more"
      ['10', '5', '27']
    • \d{1,3}   Matches 1-3 digits     Examples

              
      import re
s = "January 20, 2021"
re.findall(r'\d{1,3}', s)
      ['20', '202', '1']

      import re
s = "They spent $1,250,000 on 45,000 trees in 2021."
re.findall(r'\d{1,3}(?:,\d\d\d)+', s)
      ['1,250,000', '45,000']
    • \d{4,}   Matches at least 4 digits     Example

              
      import re
s = "10000 people gathered on January 20, 2021."
re.findall(r'\d{4,}', s)
      ['10000', '2021']
    • \d{,4}   Matches up to 4 digits     Example

              
      import re
s = "He spent $8000 in a budget of $11000."
re.findall(r'\$\d{,4}', s)
      ['$8000', '$1100']
    • \d{4}   Matches 4 digits     Example

              
      import re
s = "10000 people gathered on January 20, 2021."
re.findall(r'\b\d{4}\b', s)
      ['2021']
    • \s   Matches a whitespace (e.g. regular space, tab, newline, no-break space, thin/wide spaces)     Example

              
      import re
s = "A line with a\ttab."
re.findall(r'\s', s)
      [' ', ' ', ' ', '\t']
    • \S   Matches any non-whitespace     Example

              
      import re
s = "2 cows."
re.findall(r'\S', s)
      ['2', 'c', 'o', 'w', 's', '.']
    • \S+   Matches one or more non-whitespace     Example

              
      import re
s = "2 cows."
re.findall(r'\S+', s)
      ['2', 'cows.']
    • \s?   Matches zero or one whitespace (? indicates 0 or 1 of the preceding character or group; optional)     Example

              
      import re
s = "Temperatures: 68F, 20 C"
re.findall(r'\d+\s?[FC]\b', s)
      ['68F', '20 C']
    • \s*   Matches zero or more whitespaces (* indicates 0 or more of the preceding character or group)     Example

              
      import re
s = "John,Mary, and Paul"
re.findall(r'\S,\s*\S', s)
      ['n,M', 'y, a']
    • \s+   Matches one or more whitespaces (+ indicates 1 or more of the preceding character or group)     Example

              
      import re
s = "John,Mary, and Paul"
re.findall(r'\S,\s+\S', s)
      ['y, a']
    • \w   Matches a word character (letter or digit)     Example

              
      import re
s = '5 words: Dvořák, Москва المغرب 中国 भारत'
re.findall(r'\w+', s)
      ['5', 'words', 'Dvořák', 'Москва', 'المغرب', '中国', 'भ', 'रत']
      Note that word 'भारत' is split into 'भ', 'रत' . This is due to a vowel sign inside the Hindi word that does not count as a "word" character in regular expressions. See "Practical examples: Indian words" for how to fix this.
    • [aeiou]   Matches any character inside the square brackets     Example

              
      import re
s = "The clown laughed."
re.findall(r'[aeiou]', s)
      ['e', 'o', 'a', 'u', 'e']
    • [^aeiou]   Matches any character not inside the square brackets after initial ^     Example

              
      import re
s = "The clown laughed."
re.findall(r'[^aeiou]', s)
      ['T', 'h', ' ', 'c', 'l', 'w', 'n', ' ', 'l', 'g', 'h', 'd', '.']
    • [1-9a-z]   Matches any character in a range.     Example

              
      import re
s = "In 2021!"
re.findall(r'[1-9a-z]', s)
      ['n', '2', '2', '1']
    • .   Matches any character (except newline)     Examples

              
      import re
s = "DSCI510 from 9-5 but not 24/7"
re.findall(r'\d.\d', s)
      ['510', '9-5', '4/7']

      import re
s = "He can ran 100 meters in 10.35 seconds."
re.findall(r'\d+\.\d+', s) # \. stands for an actual period
      ['10.35']
    • +?   Repeats a character one or more times (non-greedy, i.e. as short as possible)     Examples

              
      import re
s = "He came on March 15 and left on March 22, 2021."
re.findall(r'(March.+\d+)', s)
      ['March 15 and left on March 22, 2021']

      import re
s = "He came on March 15 and left on March 22, 2021."
re.findall(r'(March.+?\d+)', s)
      ['March 15', 'March 22']
    • *?   Repeats a character zero or more times (non-greedy, i.e. as short as possible)     Examples

              
      import re
s = '<a href="https://www.usc.edu" title="University of Southern California">USC<\a>'
re.findall(r'".*"', s)
      ['"https://www.usc.edu" title="University of Southern California"']
      Returns 1 string, probably not intended.

      import re
s = '<a href="https://www.usc.edu" title="University of Southern California">USC<\a>'
re.findall(r'".*?"', s)
      ['"https://www.usc.edu"', '"University of Southern California"']
      Returns 2 strings, as intended.
    • ^   Matches the beginning of a line     Example

              
      import re
s = "10+20=30"
re.findall(r'^\d+', s)
      ['10']
    • $   Matches the end of a line     Examples

              
      import re
s = "10+20=30"
re.findall(r'\d+$', s)
      ['30']

      import re
s = "He spent $400 on a dinner for two on February 14."
re.findall(r'\$\d+', s) # \$ stands for an actual dollar sign
      ['$400']
    • \b   Matches word boundary     Examples

              The boundary marker prevents the F in Ford to match the F as an abbreviation for Fahrenheit.
      import re
s = "99F in a 1999 Ford Mustang."
re.findall(r'(\d+)\s*([CF])\b', s)
      [('99', 'F')]

      import re
s = "99F in a 1999 Ford Mustang."
re.findall(r'(\d+)\s*([CF])', s)
      [('99', 'F'), ('1999', 'F')]
    • (...)   Captures a group     Example

              
      import re
s = "68F is 20 C."
re.findall(r'(\d+)\s*([CF])', s)
      [('68', 'F'), ('20', 'C')]
    • (?:...)   Define a non-capture group     Example

              
      import re
s = "They spent $1,250,000 on 45,000 trees in 2021."
re.findall(r'\d{1,3}(?:,\d\d\d)+', s)
      ['1,250,000', '45,000']
    • cat|dog   Alternatives     Examples

              Alternatives inside a capture group:
      import re
s = "He visited us on June 27 and July 4."
re.findall(r'(June|July|August)\s+([1-9]|[12]\d|3[01])\b', s)
      [('June', '27'), ('July', '4')]

      Alternatives inside a non-capture group:
      import re
s = "He visited us on June 27 and July 4."
re.findall(r'(?:June|July|August)\s+(?:[1-9]|[12]\d|3[01])\b', s)
      ['June 27', 'July 4']
      Note that days of the month are limited to 1-31.
    • r'...'   Interprets the string as a regular expression, i.e. without interpreting \ as a Python string escape character.     Examples

              With an r prefix, the \ inside a string is taken literally, not interpreted as the start of an escape sequence.
      s1 = '\b'
s2 = r'\b'
f'{s1} has {len(s1)} character; {s2} has {len(s2)} characters.'
      '\x08 has 1 character; \\b has 2 characters.'
      The r prefix is useful for regular expressions. (Note: \x08 represents the backspace control character.)

      In LaTeX, there is a \begin tag, which in Python has to be escaped as '\\begin' because in Python, '\b' is interpreted as a backspace. In a regular expression, \begin also has to be escaped, because '\b' is interpreted as a word-boundary mark.
      import re
# string with single backslashes before begin:
s = '\\begin{document} \\begin{section}' 
re.findall('(\\begin){([^{}]+)}', s)
      []
      The above does not work (does not find anything), because Python interprets the \\begin part of a string as a 6-character string with a single \. The regular expression then interprets the resulting '\begin' as a word-boundary mark followed by the 4 letters egin . (Bad!)

      The following resolves the problem by doing a double escape. In Python, '\\\\begin' is the 7-character string \\begin (starting with two backslashes), as each '\\' is interpreted as the escape sequence for a \. The regular expression processor then interprets the \\ as an escape sequence for one literal \, which is what we intended.
      import re
s = '\\begin{document} \\begin{section}'
re.findall('(\\\\begin){([^{}]+)}', s)
      [('\\begin', 'document'), ('\\begin', 'section')]
      It works, but now the backslashes are getting out of hand, leading to "backslashitis": \\\\

      Backslashitis can be controlled by using the r'' string prefix to suppress regular Python escape ineterpretation:
      import re
s = r'\begin{document} \begin{section}'
re.findall(r'(\\begin){([^{}]+)}', s)
      [('\\begin', 'document'), ('\\begin', 'section')]
    • \1 \2   Backreferences that refer back to previously captured groups.     Examples

              Insert a space between a digit and a letter.
      import re
s = "3cats and 5dogs"
re.sub(r'([0-9])([a-z])', r'\1 \2', s)
      '3 cats and 5 dogs'
      \1 stands for the 1st captured group, \2 for the second captured group.

      Find adjacent duplicate digits.
      import regex
s = "1233456666789"
regex.findall(r'((\d)\2)', s)
      [('33', '3'), ('66', '6'), ('66', '6')]
      \1 stands for the 1st (= outer) captured group, \2 for second (= inner) captured group.

      Find adjacent duplicate words.
      import regex
s = "He told me that that was impossible because of of the theory."
regex.findall(r'((?:\pL\pM*)+)\s\1(?!\pL)', s)
      ['that', 'of']
      \1 stands for the 1st captured group. \pL is any letter. \pM any letter modifier.
    • Metacharacters .^*+?{}[]()\|$   To match a character that is also used as a metacharacter, escape it with a blackslash, etc. \. \$     Examples

              
      import re
s = "He can ran 100 meters in 10.35 seconds."
re.findall(r'\d+\.\d+', s) # \. stands for an actual period
      ['10.35']

      import re
s = "He spent $400 on a dinner for two on February 14."
re.findall(r'\$\d+', s) # \$ stands for an actual dollar sign
      ['$400']
    Lookarounds
    • (?<=...)   Positive lookbehind: left context must match the ... inside (?<=...)     Example

              
      import re
s = "On February 16, 2022, gas cost between $4.35 and $5 a gallon."
re.findall(r'(?<=\$)(\d+(?:\.\d+)?)', s)
      ['4.35', '5']
    • (?=...)   Positive lookahead: right context must match the ... inside (?=...)     Example

              
      import re
s = "I paid 50 dollars for these 2 tickets."
re.findall(r'(\d+(?:\.\d+)?)(?=\s*dollars?\b)', s)
      ['50']
    • (?<!...)   Negative lookbehind: left context must NOT match the ... inside (?<!...)     Examples

              The following example mistakenly identifies the -15 in 12-15 as a negative number:
      import re
s = "12-15"
re.findall(r'-?\d+', s)
      ['12', '-15']

      We can avoid a range "-" to be interpreted as a minus "-", by forbidding the number to be directly preceded by a digit:
      import re
s = "12-15"
re.findall(r'(?<!\d)-?\d+', s)
      ['12', '15']

      Find words ending in -man but not in -human or -woman:
      import regex
s = "chairman, demand, human, man, policeman, spokeswoman, subhuman, woman"
regex.findall(r'\pL*(?<!hu|wo)man\b', s)
      ['chairman', 'man', 'policeman']
    • (?!...)   Negative lookahead: right context must NOT match the ... inside (?!...)     Examples

              Looking for names that are not followed by Sr.:
      import re
s = "Reagan, Bush Sr. and Clinton"
re.findall(r'(\b[A-Z][a-z]{2,})\b(?! Sr\.)', s)
      ['Reagan', 'Clinton']

      Find words starting with un- but not with under-:
      import regex
s = "sunny, unable, unclear, under, underground, understand, undo, unfair"
regex.findall(r'\bun(?!der)\pL*', s)
      ['unable', 'unclear', 'undo', 'unfair']
    • More:   https://www.regular-expressions.info/lookaround.html   Lookarounds for regular expressions  
    Methods
    • search   Returns a Match-object if there is a match anywhere in the string.     Examples

              
      import re
s = "He traveled on June 16 and on July 4."
m = re.search(r'on (June|July) (\d+)', s)
m.group(0, 1, 2)
      ('on June 16', 'June', '16')
      Returns a match object for the first match. m.group(0) covers the whole match, m.group(1) the 1st captured group, m.group(2) the 2nd captured group.

      import re
s = "The temperature was 70°F."
m = re.search(r'(-?\d+(?:\.\d+)?)\s*°\s*([CF])\b', s)
m.group(0, 1, 2)
      ('70°F', '70', 'F')
    • match   Returns a Match-object if there is a match at the beginning of the string.     Examples

              
      import re
s = "From stephen.marquard@uct.ac.za Sat Jan  5 09:14:16 2008"
m = re.match(r'From (\S+@\S+).*?(200[5-8]|2010)', s)
m.group(0, 1, 2)
      ('From stephen.marquard@uct.ac.za Sat Jan 5 09:14:16 2008', 'stephen.marquard@uct.ac.za', '2008')
      Returns a match object. m.group(0) covers the whole match, m.group(1) the 1st captured group, m.group(2) the 2nd captured group.

      import re
s = "From stephen.marquard@uct.ac.za Sat Jan  5 09:14:16 2008"
re.match(r'(\S+@\S+).*?(200[5-8]|2010)', s)
      None
      No match, because the email address (\S+@\S+) is not at the beginning of the string.
    • findall   Returns a list containing all matches.     Examples

      ['10', '5', '27']

      import re

re.findall(r'(-?\d+(?:\.\d+)?)\s*°\s*([CF])\b', s)
      [('70', 'F'), ('21.1', 'C')]
    • sub   Substitute all parts of a string that match a regular expression with something else.     Examples

              The following substitution removes punctuation.

s = '• My fiancée (Mary) said: “hello”.'
regex.sub(r'\pP', '', s)
      ' My fiancée Mary said hello'

      Insert a space between a digit and a letter.
      import re
s = "3cats and 5dogs"

      '3 cats and 5 dogs'
      \1 stands for the 1st captured group, \2 for the second captured group.
    • split   Returns a list where the string has been split at each match     Example

              Split string around commas and semicolons (incl. any following spaces)
      import re
s = "apples, bananas,red onions; apple pie"
re.split(r'[,;]\s*', s)
      ['apples', 'bananas', 'red onions', 'apple pie']
    • compile   Compiles a regular expression into a reusable object. Might also speed up the code at run time.     Example

              
      import re
re_float = re.compile(r'\d+\.\d+')
(re_float.findall('$8.50 to $10.99 a pound'), re_float.findall('He is 1.88m.'))
      (['8.50', '10.99'], ['1.88'])

    Practical examples
    • Remove punctuation   including non-ASCII punctuation, but keeping non-ASCII letters     Example

              The following substitution removes punctuation.
      import regex
s = '• My fiancée (Mary) said: “hello”.'
regex.sub(r'\pP', '', s)
      ' My fiancée Mary said hello'
    • Detect adjacent duplicate words.     Example

              Find adjacent duplicate words.
      import regex
s = "He told me that that was impossible because of of the theory."
regex.findall(r'((?:\pL\pM*)+)\s\1(?!\pL)', s)
      ['that', 'of']
      \1 stands for the 1st captured group. \pL is any letter. \pM any letter modifier.
    • Numbers   integers and floating point numbers (ASCII)     Example

              
      import re
s = "100 -10C +10.5C 0 0.1 .2 3.14159 January 12-14 192.0.2.1 000 0."
re.findall(r'(?<![0-9.])[-+]?(?:[1-9][0-9]*\.[0-9]+|0?\.[0-9]+|[1-9][0-9]*|0)(?!\.?[0-9])', s)
      ['100', '-10', '+10.5', '0', '0.1', '.2', '3.14159', '12', '14', '0']
      Includes some context sensitivity to exclude things such as IP addresses (192.0.2.1) and negative numbers from ranges (12-14).
    • Numbers with commas   125,500     Example

              
      import re
s = "They spent $1,250,000 on 45,000 trees in 2021."
re.findall(r'\d{1,3}(?:,\d\d\d)+', s)
      ['1,250,000', '45,000']
    • Hybrid numbers (ASCII)   3.4 billion     Example

              Find and map hybrid numbers ("2 million") to numeric (2000000)
      import re
def combine_hybrid_number_components(tuple): # tuple: ('2', 'million')
    d = {'million':1000000, 'billion':1000000000, 'trillion':1000000000000}
    product = float(tuple[0]) * d.get(tuple[1].lower())
    return int(product) if product.is_integer() else product

s = "Sales in 2021 were $3.4 BILLION, up $200 million from the year before."
h = re.findall(r'(\d+(?:\.\d+)?)\s*((?:m|b|tr)illion)\b', s, re.IGNORECASE)
list(map(combine_hybrid_number_components, h))
      [3400000000, 200000000]
    • Indian words (with letter modifiers)   भारत     Examples

              Many Indian languages use letter modifiers (\pM) such as vowel signs and viramas that need to be added when searching for words:
      import regex
s = "नई दिल्ली से जर्मनी तक."
regex.findall(r'(?:\pL\pM*)+', s)
      ['नई', 'दिल्ली', 'से', 'जर्मनी', 'तक']

      Without letter modifiers, things fall apart.
      import regex
s = "नई दिल्ली से जर्मनी तक."
regex.findall(r'\pL+', s)
      ['नई', 'द', 'ल', 'ल', 'स', 'जर', 'मन', 'तक']
    • Simple glosser   Spanish to English     Example

              This simple glossing program translates a sentence word-by-word, without context, using a simple dictionary.
      import regex
spa_eng_dict = {"amigos": "friends", "hermana": "sister", "mi": "my", "pequeña": "little", "tiene": "has"}
s = "María (mi hermana pequeña) tiene 1000 amigos."

rest = s
result = ''
while m3 := regex.match(r'(.*?)(\pL+)(.*)', rest):   # Capture pre-word (minimal), word, post-word.
    pre_word, word, rest = m3.group(1, 2, 3)         # post-word becomes new rest for next iteration.
    result += pre_word + spa_eng_dict.get(word, word)
result += rest                                       # Add remaining text (e.g. after the last word).

result
      'María (my sister little) has 1000 friends.'
      The program uses a widely applicable idiom of regular expression, where a pattern of interest (here "\pL+" (= letters)) is preceded by a non-greedy pre-pattern string (.*?) and followed by a greedy post-pattern catch-the-rest (.*). The function preserves non-words such as spaces, punctuation, digits as well as words not covered by the dictionary (e.g. "María"). In the real world, the content of spa_eng_dict would be read in from a much larger dictionary file.

      def eng_plural(s: str) -> str:
    irreg_eng_plurals = {"foot": "feet", "mouse": "mice"}
    if irregular_plural := irreg_eng_plurals.get(s):
        return irregular_plural
    elif s.endswith('y'):
        return s[:-1] + 'ies'
    elif s[-1:] in ['s', 'x'] or s[-2:] in ['ch', 'sh']:
        return s + 'es'
    else:
        return s + 's'

      Find and map hybrid numbers ("2 million") to numeric (2000000)
      import re
def combine_hybrid_number_components(tuple): # tuple: ('2', 'million')
    d = {'million':1000000, 'billion':1000000000, 'trillion':1000000000000}
    product = float(tuple[0]) * d.get(tuple[1].lower())
    return int(product) if product.is_integer() else product
